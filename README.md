# Char

"Show some character"

This is a python script that takes an image, and converts it into an ASCII art version of itself, saved under `(image name here)_ascii.txt`.

## Dependencies

- Python 3 interpreter
- Pillow library

## Example

Transforms [this image](./visi.png) into [this text](./visi_ascii.txt) by running:
```bash
python3 ./script.py
```
and giving ./visi.png as the input path.
